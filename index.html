Cloud Sync Setup — one-time, ~5 minutes

This connects the tracker to a free Google Sheet so data saves instantly to the cloud and survives closing the browser or switching devices.

1
Create a Google Sheet
Go to sheets.google.com and create a new blank sheet. Name it anything (e.g. "2026 Promo Day Tracker").
2
Open Apps Script
In the sheet, click Extensions → Apps Script. Delete any existing code in the editor.
3
Paste this code
Copy everything in the box below and paste it into the Apps Script editor: 
const SHEET_NAME = "Data";

function doGet(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sheet = ss.getSheetByName(SHEET_NAME);
  if (!sheet) sheet = ss.insertSheet(SHEET_NAME);
  const cell = sheet.getRange("A1").getValue();
  const payload = cell ? { state: cell } : { state: null };
  return ContentService
    .createTextOutput(JSON.stringify(payload))
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  let sheet = ss.getSheetByName(SHEET_NAME);
  if (!sheet) sheet = ss.insertSheet(SHEET_NAME);
  const body = JSON.parse(e.postData.contents);
  if (body.action === "save") {
    sheet.getRange("A1").setValue(body.state);
    // Write timestamp
    sheet.getRange("B1").setValue(new Date().toLocaleString());
  }
  return ContentService
    .createTextOutput(JSON.stringify({ ok: true }))
    .setMimeType(ContentService.MimeType.JSON);
}
4
Deploy as Web App
Click Deploy → New deployment. Set type to Web app. Set Execute as to Me and Who has access to Anyone. Click Deploy, authorize it, then copy the Web App URL.
5
Paste the URL into the tracker HTML file
Open the Player_Shoot_Tracker.html file in a text editor. Find this line near the top of the <script> section:
const GAS_URL = "";
Replace it with your URL:
const GAS_URL = "https://script.google.com/macros/s/YOUR_ID/exec";
Save the file. Anyone who opens it will now sync to the same Google Sheet.
6
Done. The sync indicator in the top right will show ✓ Cloud saved when working. Your Google Sheet will also show the raw data in cell A1 and the last save time in B1.
Progress is saved automatically in this browser on this device. Use Export CSV to combine data from multiple devices.
